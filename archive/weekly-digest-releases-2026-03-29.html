Release 2026-02-08

📅 2026-02-17 		View Release ↗

This release emphasizes the retirement of Windows Server 2019 and introduces Kubernetes patch versions.

❌ Breaking Changes

  • Windows Server 2019 will be unsupported after March 1, 2026.

🔑 Key Features

  • Kubernetes patch versions 1.34.2, 1.33.6, and 1.32.10 are now available.
  • Managed GPU profiles are introduced for better resource management.

💡 Good to Know

  • Transition to Windows Server 2022+ is necessary before retirement.
  • Refer to the AKS-Release-Tracker for monitoring release status.

Release 2026-01-04

📅 2026-01-09 		View Release ↗

The release introduces Ubuntu 24.04 and highlights the end of support for Azure Linux 2.0.

❌ Breaking Changes

  • Azure Linux 2.0 will no longer receive security updates after November 30, 2025.

🔑 Key Features

  • Ubuntu 24.04 will be the default OS starting in Kubernetes v1.35.
  • CIS Kubernetes Benchmark results are now published for AKS.

💡 Good to Know

  • Migrate to a supported Azure Linux version to avoid issues.
  • Detailed CIS reports are available in the documentation.

Release 2025-09-21

📅 2025-09-26 		View Release ↗

The release includes deprecation notices for Kubernetes version 1.31 and introduces version 1.34 in preview.

❌ Breaking Changes

  • Standard support for Kubernetes version 1.31 will end on November 1, 2025.
  • InPlaceOrRecreate feature gate for Vertical Pod Autoscaling will be enabled in version 1.34.

🔑 Key Features

  • Kubernetes version 1.34 is now available in preview.
  • Vertical Pod Autoscaler components will have 2 replicas by default in version 1.34.

💡 Good to Know

  • Refer to the version support policy for upgrade guidance.
  • Migration to Long Term Support is recommended for version 1.31.
